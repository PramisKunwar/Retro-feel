# Retrofy Web — Chrome Extension

A **Manifest V3 Chrome Extension** that transforms any modern website into a retro terminal-style interface, inspired by 1980s CRT monitors, amber/green monochrome terminals, and classic DOS/UNIX aesthetics — while preserving full usability.

## 🎮 Usage

| Action | How |
|---|---|
| Toggle retro mode | Click the extension icon → flip the switch |
| Change theme | Click the extension icon → select Amber / Green / White |
| Open command palette | `Ctrl + Shift + K` (or `Cmd + Shift + K` on Mac) |
| Disable for current site | Click the extension icon → toggle "This Site" off |
